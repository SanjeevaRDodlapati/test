
### Checking current version and it's PATH
```console
python --version
```
```console
which python3
```

```console
type python3
```

```console
echo $PATH
```
##### setting python path in .bash_profile
```console
nano ~/.bash_profile
```
Add the path variable to .bash_profile
```console
PATH="enter python path here"
```

Aiasing python
```console
nano ~/.bash_profile
alias python=python3
alias pip=pip3
```

Check
```console
type python
type python3
type pip
type pip3
```

Checking executable python
```console
python 
>>> import sys
>>> sys.executable
```
