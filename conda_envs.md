## Conda Envs:
Check existing envs
```
conda info --envs # or
conda env list
```

Create env
```
conda create --name myenv
```

Create with specific python version
 ```
 conda create -n myenv python=3.6 #or
 coda create -n myenv python
 ```
 ```
 conda activate myenv
 ```
 
 Remove/deactivate env
 ```
 conda deactivate
 ```
 
 ```
 conda env remove -n myenv
 ```
